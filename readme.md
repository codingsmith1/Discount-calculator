# 🧮 Discount Calculator

A clean and minimal web app that calculates the final discounted price and total savings based on user input.

---

## 📸 Preview

> A white card UI centered on a light grey background with rounded input fields and a blue Calculate button.

---

## ✨ Features

- Input field for **Original Price**
- Input field for **Discount Percentage**
- Click **Calculate** to instantly compute:
  - ✅ Discounted Price
  - ✅ Amount Saved
- Results displayed in highlighted blue text

---

## 🛠️ Technologies Used

| Technology | Usage |
|------------|-------|
| **HTML5** | Page structure and form inputs |
| **CSS3** | Card layout, rounded inputs, button styling |
| **JavaScript** | Discount calculation logic and DOM updates |

---

## 🧮 How It Works

```
Discounted Price = Original Price - (Original Price × Discount% / 100)
Amount Saved     = Original Price - Discounted Price
```

---

## 📱 Responsive Design

Tested on:
- S — 320px
- M — 375px
- L — 435px
- Tablet — 768px

---

## 🚀 Getting Started

1. Open `index.html` in any modern browser
2. Enter the original price
3. Enter the discount percentage
4. Click **Calculate**

> No dependencies or build tools required!
